Transcribe and summarise audio

Upload a WAV recording or record live from your microphone. The audio is transcribed using Microsoft MAI‑Transcribe‑1.5, then you can summarise the result with an AI language model.

Before you start

  • If uploading a file, it must be a WAV audio file.
  • If recording from the microphone, you will need to allow your browser to access it.
  • Do not upload audio that contains sensitive or personal information.
